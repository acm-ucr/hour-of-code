Hour of Code

December 2nd, 10AM - 2PM

Winston Chung Hall, UC Riverside

Hour of Code is a global movement to increase access to Computer Science education. Hour of Code at UCR is hosted by ACM@UCR to teach over 150 K–12 students how to program through workshops, tutorials, and 1-on-1s.

About Hour of Code at UCR

Programming Fundamentals

Learn the basics of programming. Take what you learn from this series and apply it to math, biology, and many more subjects.

Web Development

Learn the basics of HTML and CSS to create websites.

Game Development

Learn how to make video games!

Art

Learn how to create masterful art pieces with programming.
